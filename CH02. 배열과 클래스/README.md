#### 2.1 배열(array)
- 같은 데이터형의 요소들이 동일한 크기로 순서를 갖고 나열되어있는 집합
- **여러 개의 동일한 자료형의 데이터를 한꺼번에 만들 때 사용**
- 배열이름[배열의 크기] = {값};
- int arr[row][col]; / col size 생략 불가
- <인덱스, 요소> 쌍의 집합. index는 0부터 시작
- 인덱스를 사용하여 요소에 직접 접근 가능


```
# 참고 포인터와 배열
*ptr = &data
arr[i] = *(arr+i)
```
